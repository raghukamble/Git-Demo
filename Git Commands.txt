git log -p
this command will tell what exactly added and removed from the file or directory

git log -p -2
It will show only 2 commits

git log --stat
Gives short summary of commits

git log --pretty=oneline
Gives all commits in single liner

git log --pretty=short
gives commit detials in short

git log --pretty=full
gives commit detials with full details

git log --since=2.month

--------------------------------------------------------------------------------------------------------------------------------------------------
git log --pretty-format: "%h -- %an"
Gives commit hash id and author name(an)
